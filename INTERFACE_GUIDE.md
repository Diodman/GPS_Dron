# 🚁 Руководство по новому интерфейсу системы управления дронами

## 🎨 Обзор интерфейса

Новый интерфейс представляет собой современное веб-приложение с красивым дизайном и интуитивно понятным управлением. Интерфейс разделен на три основные вкладки:

### 📋 **Вкладка "Заказы"**
- **Создание заказов** с выбором типа дрона и приоритета
- **Просмотр активных заказов** с возможностью отмены и изменения
- **Управление промежуточными точками** для сложных маршрутов

### 🚁 **Вкладка "Дроны"**
- **Мониторинг флота дронов** в реальном времени
- **Детальная информация** о каждом дроне (батарея, здоровье, статус)
- **Визуальные индикаторы** - разные фигуры для разных типов дронов
- **Возможность ремонта** сломанных дронов

### ⚙️ **Вкладка "Настройки"**
- **Управление базой** и инвентарем дронов
- **Настройка бесполётных зон** и станций зарядки
- **Загрузка карт городов**

## 🎯 Ключевые функции

### 🌤️ **Погодная панель**
- **Температура, влажность, скорость ветра, видимость**
- **Автоматическое обновление** каждые несколько секунд
- **Влияние на полеты** - система учитывает погодные условия при планировании маршрутов

### 🎨 **Визуальные индикаторы дронов**
- **🔴 Круг** - Грузовые дроны (cargo)
- **🔵 Квадрат** - Операторские дроны (operator)  
- **🔺 Треугольник** - Уборочные дроны (cleaner)

### 📊 **Статистика в реальном времени**
- **Количество дронов** в системе
- **Активные заказы** и их статусы
- **Текущий город** и загруженная карта

### 🗺️ **Интерактивная карта**
- **Отображение дронов** с их маршрутами
- **Бесполётные зоны** (красные прямоугольники)
- **Станции зарядки** (зеленые кружки)
- **История полетов** (пунктирные линии)

## 🎮 Управление

### 🖱️ **Работа с картой**
- **Клик** - выбор координат для заказов
- **Shift + перетаскивание** - создание бесполётной зоны
- **Alt + клик** - добавление станции зарядки
- **Ctrl + правый клик** - удаление ближайшей станции

### 📝 **Создание заказов**
1. Выберите **тип дрона** (или оставьте автоматический выбор)
2. Укажите **тип заказа** (доставка, съёмка, работа, инспекция)
3. Введите **адреса или координаты** откуда и куда
4. Добавьте **промежуточные точки** при необходимости
5. Установите **приоритет** (обычный, высокий, критический)
6. Нажмите **"Создать заказ"**

### 🔧 **Управление дронами**
- **"Найти"** - центрировать карту на дроне
- **"Детали"** - открыть подробную информацию
- **"Ремонт"** - исправить сломанный дрон

### ⚙️ **Настройки системы**
- **База и инвентарь** - координаты базы и количество дронов каждого типа
- **Бесполётные зоны** - создание и удаление запрещенных зон
- **Станции зарядки** - добавление и удаление станций
- **Город** - загрузка карты нового города

## 🚀 Запуск системы

1. **Запустите сервер:**
   ```bash
   python api_server.py
   ```

2. **Откройте браузер:**
   ```
   http://localhost:8000
   ```

3. **Настройте базу:**
   - Перейдите на вкладку "Настройки"
   - Укажите координаты базы
   - Установите количество дронов каждого типа
   - Нажмите "Сохранить базу"

4. **Загрузите карту города:**
   - Введите название города
   - Нажмите "Загрузить карту"

5. **Создайте первый заказ:**
   - Перейдите на вкладку "Заказы"
   - Заполните форму создания заказа
   - Нажмите "Создать заказ"
